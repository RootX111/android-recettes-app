# Android Recettes App

Application Android développée pour le TP STIC.

## Fonctionnalités

- Connexion utilisateur
- Lecture d’un fichier JSON
- Affichage des recettes dans une ListView
- Affichage des ingrédients
- Enregistrement dans SQLite

## Technologies

- Kotlin
- Android Studio
- JSON
- SQLite
